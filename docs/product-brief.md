# HotLoop Product Brief

## One-liner

HotLoop is a local-first agent cockpit and toolroom that helps a creator and an external agent detect time-sensitive AI signals, choose high-value topics, preserve evidence, generate reviewable article packages, render final HTML, and create publishing drafts.

## Target User

The first user is a solo AI / technology content creator who tracks fast-moving sources such as X, GitHub Trending, Hacker News, official AI product updates, and selected papers.

The executor is the agent. HotLoop is the operating surface where the human instructs the agent, inspects tool calls, answers decision requests, and resumes durable runs.

## Core Jobs

1. See the latest high-value AI and tech signals, with the freshest and highest-impact items first.
2. Separate explosive X posts, GitHub trends, community debates, official updates, and paper signals into clear lanes.
3. Turn a selected topic into an evidence-backed article package.
4. Generate images and final HTML artifacts without polluting the finished artifact folder.
5. Prepare publishing drafts while keeping final human review in the loop.
6. Interact with the agent through durable sessions, messages, commands, tool events, and human decision checkpoints.

## Non-goals

- Do not become a generic AI writing tool.
- Do not become the hidden executor; the agent remains the executor.
- Do not replace the local content vault with a database in the first version.
- Do not auto-publish to external platforms.
- Do not require multi-user SaaS infrastructure for the initial product.
- Do not hide source evidence behind opaque search results.
